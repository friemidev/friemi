# Avalon Color Notes

The Avalon helper should feel light, social, and part of Friemi rather than like a heavy medieval game skin.

## Palette Mapping

| Purpose | Color |
|---|---|
| Primary forest | `#156240` |
| Soft sage | `#8AB68E` |
| Warm coral | `#F09182` |
| Cream background | `#FEFFF9` |
| Paper background | `#F1F2EC` |
| Warm chip | `#FFF5E6` |
| Neutral line | `#D6D5B2` |
| Ink | `#1D1D1B` |

## Usage Rules

- Use forest green for primary action, table structure, and good-side clarity.
- Use coral for attention, danger, rejection, and shadow-side accents.
- Avoid saturated purple, neon blue, dark fantasy gradients, blood effects, and realistic weapons.
- Keep role icons readable at 48px and friendly at 96px.
- Use SVG with no baked-in language text wherever possible.
