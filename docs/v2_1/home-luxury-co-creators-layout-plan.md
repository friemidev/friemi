# v2.1 首页布局规划：共创主理人主推版

## 页面目标

v2.1 首页不再只作为普通入口页，而是承担 Friemi 品牌第一印象和共创主理人招募主推入口。页面核心内容围绕 `/co-creators` 展开，让用户先感受到 Friemi 的生活方式、组织活动的专业感和个人 IP 成长价值，再自然进入活动发现、组队大厅或共创主理人页面。

首页方向：

```text
高星级酒店式视觉质感 -> 共创主理人价值叙事 -> 活动内容作为节奏休息区 -> 引导进入 /co-creators
```

## 设计基调

参考高星级酒店官网的质感，但不照搬酒店预订页。

关键词：

```text
克制、高级、真实、留白、服务感、城市生活方式、可积累的社群
```

视觉原则：

- 使用真实感图片或视频作为情绪核心，不使用廉价渐变或模板插画
- 页面以大图、留白、细线、低饱和暖色和高级 serif 标题建立质感
- 文案少而准，不做密集功能说明
- 内容区要像 editorial magazine，而不是 SaaS landing page
- CTA 数量控制，主路径始终指向 `/co-creators`
- 活动卡片作为节奏转换，不喧宾夺主

## 页面结构

### 1. 沉浸式 Hero：把 Friemi 定位成城市活动社群工具

目标：

用户打开首页后，先看到一个高级、真实、有生活方式气质的 Friemi，而不是功能堆叠。

结构：

- 全宽大图或短视频背景
- header 轻量覆盖在画面上，降低边框和框感
- 中央或偏左放一句主文案
- 主 CTA 指向 `/co-creators`
- 次 CTA 指向活动大厅或组队大厅

建议文案方向：

```text
让零散的小局，变成可积累的活动社群。
```

辅助文案：

```text
Friemi 帮助你更好转发活动、管理报名、沉淀参与者关系，并把个人 IP 变成更清晰的主理人品牌。
```

CTA：

- 主按钮：成为共创主理人
- 次按钮：先看看活动

动效：

- 首屏图像轻微 slow zoom
- Logo / 主标题 / CTA 分层淡入
- 下滑提示保持极轻，不使用大箭头或强引导

### 2. 品牌宣言区：承接 `/co-creators` 的核心一句话

目标：

在 Hero 后用更安静的版面解释 Friemi 对主理人的价值。

结构：

- 大面积暖白背景
- 居中大标题
- 一段不超过 3 行的正文
- 可加入非常轻的城市线稿或细线装饰，但不能抢内容

核心文案：

```text
让你原本零散的小局，变成更好传播、更容易报名、更方便复用、更能沉淀人气，并且帮助你的个人 IP 更加品牌化的活动社群。
```

布局参考：

- 类似酒店官网下滑后的 intro section
- 文案居中，但宽度严格控制
- 不放卡片，不放太多 badge

### 3. 共创主理人价值路径：七个关键词变成高级信息流

目标：

把 `/co-creators` 的七个关键词拆成首页可快速扫读的路径，但不要做成 7 张大卡片。

关键词：

```text
好转发｜更专业｜易报名｜易复用｜能沉淀｜强 IP｜有身份
```

结构建议：

- 桌面端：左侧固定大图或主理人场景，右侧纵向价值列表
- 移动端：标题 + 3 到 4 条优先价值，剩余内容用轻量展开或下方继续叙事
- 每条价值用数字、细线和短句，不使用厚重卡片

示例排布：

```text
01 好转发
活动链接清晰完整，方便分享到微信群、朋友圈、小红书和 Instagram。

02 更专业
时间、地点、人数、费用和规则统一展示，参与者一眼看懂。

03 易报名
减少私聊、接龙和人工统计，更方便收集和管理参与者。
```

动效：

- 滚动进入时，数字和标题轻微 stagger reveal
- 图片轻微 parallax
- 不使用夸张旋转或弹跳

### 4. 活动卡片休息区：在叙事中插入真实内容

目标：

避免整页都是招募文案，让用户在阅读过程中看到 Friemi 真实活动内容，形成“这不是概念页，而是可以马上使用的产品”的感觉。

位置：

建议放在价值路径之后、个人 IP 叙事之前，作为页面中段的节奏休息区。

内容：

- 展示最近活动或精选活动
- 可以混合活动和组局，但视觉上要轻，不抢共创主线
- 标题可以是：

```text
最近正在发生的灵感
```

或：

```text
从一次小局开始
```

交互：

- 桌面端展示 3 张活动卡片，左右留出前后卡片露出一点边缘
- 移动端展示 1.15 张卡片宽度，暗示可滑动
- 支持左右按钮切换
- 每隔几秒自动切换一次
- 用户手动点击或拖动后，暂停自动切换一段时间
- 鼠标 hover / 触摸按住时暂停自动切换
- 卡片切换应平滑，不要跳动或高度变化

自动切换规则：

```text
默认间隔：5-6 秒
手动操作后暂停：8-12 秒
低动效偏好：关闭自动切换，只保留左右按钮
不足 2 张卡片：不显示左右按钮，不启用自动切换
```

卡片视觉：

- 使用当前活动卡片数据，但首页样式可以更 editorial
- 图片比例稳定，建议 4:3 或 5:4
- 标题最多两行
- 时间地点保留，但弱化状态徽章数量
- 不放太多按钮，点击整卡进入详情

### 5. 个人 IP 叙事区：从组织小局到形成主理人品牌

目标：

把共创主理人的情绪价值说完整，让首页不只是功能介绍。

结构：

- 大图 + 文案分栏
- 文案聚焦“别人会更容易记住你”
- 使用类似酒店官网的 editorial 标题和正文节奏

文案素材：

```text
你是那个长期组织读书会的人。
你是那个经常办高质量桌游局的人。
你是那个很会组织城市散步和探店活动的人。
你是那个能把大家聚在一起的人。
```

布局：

- 桌面端左右分栏，图片可略大于文字
- 移动端图片在上，文案在下
- 每一句可以做轻微分行，不放进卡片

### 6. 平台身份区：让共创身份变成明确收束

目标：

把“拥有平台共创身份”作为首页下半部分的明确利益点，引导进入 `/co-creators`。

结构：

- 深色或暖深色背景带来节奏变化
- 标题聚焦身份感
- 说明早期认证、优先展示、产品共创、未来合作机会
- CTA 再次指向 `/co-creators`

注意：

- 不要做成夸张会员卡
- 不要堆过多徽章
- 重点是身份、信任和长期参与感

### 7. 结尾 CTA：简洁收束

目标：

让用户读完后有明确下一步。

结构：

- 一句总结
- 主 CTA：成为共创主理人
- 次 CTA：浏览活动

建议总结：

```text
让零散的小局，变成可积累的活动社群；让你的个人 IP，变成更清晰的主理人品牌。
```

## 动效规划

需要有“滑动结构变化感”，但保持高级克制。

可采用：

- Hero 背景 slow zoom
- 标题和 CTA stagger reveal
- section 进入时轻微上移淡入
- 大图 parallax，但幅度小
- 活动卡片 carousel 自动切换
- 桌面端部分图文区可以使用 sticky image，右侧文案滚动切换
- 价值列表滚动时逐条亮起

避免：

- 大量弹跳动画
- 旋转、飞入、强缩放
- 每个模块都动画
- 移动端复杂 parallax 导致卡顿

低动效支持：

- `prefers-reduced-motion` 下关闭自动轮播、parallax 和 slow zoom
- 保留静态布局和左右按钮

## 移动端规划

移动端不是桌面端缩小版。

移动端优先级：

1. 首屏看到品牌主张和主 CTA
2. 立刻知道 `/co-creators` 是主推入口
3. 活动卡片休息区可以横向滑动
4. 文案分段短，不出现长段大块文字
5. 图片高度稳定，避免首屏被图片完全占满

移动端结构：

```text
Hero 半屏到 70vh
核心一句话
3 条价值摘要
活动卡片横滑休息区
个人 IP 叙事
共创身份 CTA
Footer
```

## 桌面端规划

桌面端需要更强结构变化：

```text
全屏 Hero
大留白品牌宣言
sticky 图文价值路径
活动 carousel 休息区
大图 + 个人 IP 文案
深色身份区
结尾 CTA
```

桌面端重点：

- 大图比例要稳定
- 每个 section 高度有节奏差异
- 不要所有内容都居中堆叠
- 活动卡片区作为横向节奏变化

## 素材需求

优先需要：

- 1 张首页 Hero 横图或短视频
- 3-5 张巴黎生活方式氛围图
- 1 张偏“主理人 / 组织者 / 朋友聚会”的人物场景图
- 活动卡片可以先使用真实活动数据
- 如果没有足够素材，可以先使用现有活动封面和品牌图做第一版

素材风格要求：

- 真实、温暖、精致
- 避免暗到看不清
- 避免过度 stock 感
- 避免只有空景，最好有人、动作、氛围或可想象的社交场景


```text

```

文件名使用小写英文、数字和连字符，避免中文、空格和特殊符号。

### 1. Hero 主视觉

建议文件名：


可选视频文件名：


## 素材优先级

如果不能一次准备全部素材，优先级如下：

```text
P0：Hero 主视觉、移动端 Hero 备用裁切图、共创主理人场景图
P1：朋友同行氛围图、活动内容休息区背景图、平台身份区背景图
P2：Open Graph / 分享图、轻量纹理图
```

最低可开始实现的素材组合：

```text
1 张 Hero 横图
1 张移动端 Hero 竖图
1 张共创主理人场景图
```

其余区块可以先使用现有活动封面、品牌图或纯色 editorial 布局占位。

## 素材验收标准

- 图片不能糊，桌面端 1440px 首屏看起来仍然清晰
- 不能有明显水印、版权标识或第三方 logo
- 不使用未经授权的人物正脸特写
- 画面里不能有和 Friemi 调性冲突的广告牌、杂乱文字或强商业品牌
- Hero 图必须能同时适配桌面端标题叠加和移动端裁切
- 所有素材压缩后仍应保持质感，避免过度压缩产生色块
