# v2.4 聊天图片与表情 Checklist

## 背景

私聊目前只能发送文字和基础 emoji。真实聊天里经常需要发现场照片、截图、表情图或只发一张图片，因此本分支补齐聊天图片能力，并保持输入栏轻量。
